# 메모리 단편화
어떠한 프로그램을 실행할 때, 컴퓨터에서 프로그램들을 메모리 공간에 연속적으로 할당하게 된다. 
만약 여러 프로그램들이 메모리에 할당되고 해제되는것이 반복되면 메모리 공간이 조각조각 나뉘어 메모리가 충분함에도 메모리를 할당하는 것이 불가능한 상태가 된다.

메모리 단편화를 해결하는 방법으로 **페이징**과 **세그멘테이션**이 있다.
